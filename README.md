# NODE.js

## Commands

- **create node.js _App_**  in project dir :

```bash
npm install node
```

- **Upgrade node.js** to the latest _version_ :

```bash
npm install n -g \
n latest
```

- **Runing node.js _App_** simply type :

```bash
node App.js
```
